# Tbhat Workspace

This repository provides a PyTorch Lightning deep learning pipeline designed to classify GNSS radio interference signals into nine distinct categories. At its core is VisionSignalNet, a custom 2D ResNet architecture with stochastic depth that processes raw 2D signal matrices (loaded from .npy files) by treating them as single-channel grayscale images. The data pipeline automatically manages train/test splits from text files, applies real-time Gaussian noise augmentation for model regularization, and computes inverse class weights to handle dataset imbalances. Training and evaluation are fully automated, featuring learning rate scheduling, early stopping, and comprehensive metric logging—including auto-generated confusion matrices and per-class accuracy reports—directly to Weights & Biases (WandB). To execute the model, simply update the ROOT_DIR and associated data paths in the execution block to match your environment, and run the script from the command line to initiate GPU-accelerated training and testing.
