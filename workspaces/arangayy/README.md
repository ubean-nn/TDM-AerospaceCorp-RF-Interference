# Arangayy Workspace
Contains initial baseline model prototyping, data loaders (Model.py, Model.ipynb), and basic GPU capability testing scripts (havegpu.py, rungpu.sh).

#Model.ipynb:
This is a file that is used for testing out the barebones of the algorithm as well as for short training sessions. This is not advised for use for full training.
This file includes a model that uses CNN and LSTM combined architecture, as well as data loading and testing code. 

#Model.py
This file is meant for use for long epoch training sessions and to be run on GPU hardware. Weights are suggested to be stored and reused in Model.ipynb for easier use in the future.
