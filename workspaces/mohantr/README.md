# Mohantr Workspace
Primarily dedicated to data augmentation experimentation (augmentation.ipynb) built to tackle extreme class imbalances, alongside baseline model setups and visualizers. 
The radiomana file represents the initial experimentation phase of the project. In this stage, a ResNet-18 architecture was implemented and trained, achieving relatively strong accuracy despite the dataset limitations. This phase helped validate the feasibility of the task and provided insight into how deep convolutional models respond to the given data distribution. It also served as a foundation for later refinement, including tuning hyperparameters, adjusting training procedures, and integrating augmentation pipelines.
The visualization file played a critical role in early-stage data understanding. It was used to explore class distributions, identify imbalance severity, and inspect sample images across categories. These visual diagnostics informed key design decisions, such as which classes required augmentation and what types of transformations would be most appropriate. Additionally, visualization tools were used to qualitatively assess model predictions and failure cases, helping to diagnose where the model struggled and why.
